# Changelog

All notable changes to AgentMarKB will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-02-22

### Added
- Full article content extraction via Mozilla Readability
- HTML to Markdown conversion via Turndown with GFM support
- Obsidian-compatible output (YAML frontmatter + folder structure)
- Native host `create_bookmark` and `check_exists` actions
- Content extraction status indicator in popup
- Migration script for existing bookmarks
- Python requirements.txt for migration dependencies
- Rebranded to AgentMarKB
- Open-source community files (LICENSE, CONTRIBUTING, etc.)
- GitHub Pages landing site
- New branded logo and icons

## [1.0.0] - 2026-01-12

### Added
- Initial release
- Multi-platform content extraction (X, LinkedIn, Substack, generic web)
- YAML knowledge base with author/content/topic management
- Duplicate detection with URL normalization
- Native messaging host for local file access on macOS
- Topic suggestion and management
- Options page with connection testing
