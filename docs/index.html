AgentMarKB

Turn your browser into an Agentic AI Knowledge Base. Extract full web articles as Obsidian-compatible Markdown for RAG pipelines and AI agent swarms.

Get Started View on GitHub
📄

Full Content Extraction

Mozilla Readability strips away ads and clutter. Turndown converts clean HTML to GitHub Flavored Markdown. Every article saved in its full, readable form.

📎

Obsidian-Native Output

YAML frontmatter, structured folders, tags, and metadata. Works seamlessly with Dataview queries, graph view, and your existing Obsidian workflow.

🤖

Agent-Ready Knowledge

Structured Markdown with rich metadata designed for RAG pipelines, AI agent swarms, and agentic retrieval. Your knowledge base compounds over time.

How It Works

1

Browse

Visit any article, tweet, or post. Click the AgentMarKB icon in your browser toolbar.

2

Extract

Content is automatically extracted, cleaned, and converted to Markdown with rich metadata.

3

Feed Your Agents

Structured documents land in your Obsidian vault, ready for RAG ingestion and agentic AI workflows.

Get Started in 5 Steps

# Clone the repo
git clone https://github.com/carlotorniai/agentmarkb.git
cd agentmarkb

# Load in Chrome: Extensions → Developer mode → Load unpacked

# Install the native messaging host
cd native-host && ./install.sh

# Configure: Settings → set KB path → Test Connection

# Save your first article!
