Native Host Setup

The native messaging host is required to read and write your local YAML file. Follow these steps to install it:

1

Open Terminal

Open the Terminal application on your Mac.

2

Navigate to extension folder

Navigate to the native-host folder within the extension directory:

cd /path/to/extension/native-host
3

Run install script

Execute the installation script:

./install.sh
4

Enter extension ID

When prompted, enter your Chrome extension ID. You can find it on the chrome://extensions/ page after loading the extension.

About

KB Manager saves articles, tweets, and posts to your Obsidian-compatible knowledge base. Full article content is extracted and converted to Markdown, stored as individual document folders in AI_KB/08_bookmarked_content/ following the KB convention (meta.yaml + assets/ + canonicals/). Metadata is also indexed in your YAML file for backward compatibility.
